# Bonvenuto Site Constitution

## Proposito

Servir como constituicao do projeto. Travar o que este repositorio e, o que ele nao e, e quais limites nao podem ser rompidos.

## Quando usar

- Usar ao iniciar qualquer mudanca estrutural no site.
- Usar ao revisar propostas de novas paginas, secoes, CTAs ou navegacao.
- Usar quando houver risco de reintroduzir conceitos do projeto antigo.
- Usar antes de aceitar pedidos que possam expandir escopo juridico ou tecnico.

## Quando nao usar

- Nao usar para refinamento visual pequeno quando o escopo ja estiver claro.
- Nao usar para tarefas de backend, automacao ou banco de dados, porque isso esta fora do projeto.

## Instrucoes praticas

- Tratar este repositorio como um site front-end canonico.
- Manter o WhatsApp como CTA dominante de conversao para atendimento humano.
- Manter formularios apenas como apoio eventual. Nao transformar formulario no fluxo principal.
- Nao criar backend, APIs, auth, painel, CRM, Supabase, intake engine, scoring, OCR, RAG, chatbot, handoff, automacoes externas ou qualquer camada operacional.
- Manter o escopo juridico fechado em: pensao alimenticia, pensao atrasada, revisao de pensao, paternidade e pensao, alimentos gravidicos, guarda e convivencia.
- Nao abrir divorcio, inventario, heranca, criminal, previdenciario, trabalhista, empresarial ou outras areas.
- Priorizar mobile-first em layout, hierarquia, densidade e CTA.
- Manter linguagem premium, clara, sobria e humana.
- Refinar o sistema atual antes de propor reinvencao completa.
- Sinalizar explicitamente qualquer pedido que fuja desse enquadramento antes de implementar.
- Se um pedido conflitar com esta skill, recusar a deriva e propor uma alternativa dentro do escopo.

## Checklist rapido

- O trabalho continua front-end only?
- O WhatsApp continua como CTA principal?
- A experiencia mobile continua obrigatoria, e nao secundaria?
- A mudanca respeita as 6 areas juridicas aprovadas?
- Nenhuma camada operacional do projeto antigo voltou por engano?
- A pagina continua premium, clara e sobria?
